# Hi, I'm Chris.

I create usable, responsive, future-friendly sites and apps with HTML5, CSS3, JavaScript, Haml, Sass, BEM, OOCSS, RWD, Markdown, PHP, Gulp, Grunt, Bower, npm, Jekyll, Vim, and Git.
